¿Qué es una página web?

Una página web es un documento digital que puede visualizarse mediante un navegador como Chrome, Firefox o Edge.

¿Qué es un sitio web?

Un sitio web es un conjunto de páginas web conectadas entre sí. Por ejemplo, una tienda online o una página educativa.

¿Cómo funciona Internet?

Internet es una enorme red mundial de ordenadores conectados que intercambian información continuamente.

Clientes y servidores

El cliente suele ser nuestro navegador. El servidor es el ordenador que almacena la página web.

Dirección IP

La dirección IP es un número único que identifica un dispositivo en Internet. Ejemplo: 192.168.1.1

DNS

El DNS funciona como una agenda telefónica. Transforma nombres como www.google.com en direcciones IP.

HTTP y HTTPS

HTTP es el protocolo que permite transferir páginas web. HTTPS añade cifrado para proteger la información.

Transferencia de datos

Cuando accedemos a una página web, nuestro navegador envía una petición al servidor. El servidor responde enviando los archivos HTML, CSS, imágenes y otros recursos.
