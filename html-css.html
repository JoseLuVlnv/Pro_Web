¿Qué es HTML?

HTML es el lenguaje que estructura el contenido de una página web.

Etiquetas HTML

  • <html>: contiene toda la página.
  • <head>: información interna.
  • <body>: contenido visible.
  • <h1> a <h6>: títulos.
  • <p>: párrafos.
  • <img>: imágenes.
  • <a>: enlaces.
  • <ul> y <li>: listas.

Ejemplo HTML

<h1>Hola Mundo</h1>
<p>Mi primera página web.</p>

¿Qué es CSS?

CSS es el lenguaje que permite dar estilo y diseño a una página web. Con CSS podemos modificar colores, tamaños, márgenes y mucho más.

Ejemplo CSS

h1{
    color: blue;
}

p{
    font-size:18px;
}
