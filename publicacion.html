GitHub

GitHub es una plataforma que permite almacenar proyectos de programación utilizando Git.

GitHub Pages

GitHub Pages permite publicar páginas web de forma gratuita directamente desde un repositorio.

Cómo publiqué mi proyecto

  1. Creé un repositorio en GitHub.
  2. Subí todos los archivos HTML, CSS e imágenes.
  3. Accedí a Settings → Pages.
  4. Seleccioné la rama main.
  5. GitHub generó una URL pública para mi web.

Ventajas

  • Publicación gratuita.
  • Fácil actualización.
  • Accesible desde cualquier lugar.
